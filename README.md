# 프로젝트 소개
Spring boot, Vue3.js 프레임워크를 활용하여 실 서비스 목적으로 개발중인 Blog Project 입니다.

# 기능 (구현중)
* OAuth2.0 login
* Jwt
* 블로그 카테고리
* 블로그 포스팅
* 포스팅 댓글
* ...

# 개발 환경
* OS : Windows10
* IDE : IntelliJ, VSCode
* 구성 : Spring Boot 3, Spring Data JPA, Spring Security 6, Jwt, Vue3.js, H2, PostgreSQL, Typescript, Gradle, AWS or GCP, Docker
